# Thermal Comfort Monitoring Dashboard

## Overview

A real-time thermal comfort monitoring dashboard developed during an internship project at BMKG. The system collects environmental sensor data, stores it in a time-series database, and visualizes key thermal comfort indicators through an interactive dashboard.

## Project Background

Monitoring thermal comfort conditions is important for understanding environmental quality and supporting data-driven decision making. This project focuses on building a real-time monitoring system that enables users to track environmental conditions and Thermal Humidity Index (THI) values through a web-based dashboard.

## Features

- Real-time sensor data monitoring
- Thermal Humidity Index (THI) visualization
- Interactive charts and metrics
- Time-series data storage and retrieval
- Web-based dashboard interface

## Technologies Used

- JavaScript
- Node.js
- HTML
- InfluxDB
- IoT Sensor Data
- Data Visualization
